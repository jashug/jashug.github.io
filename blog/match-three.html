Match-3 Game

A match 3 game, like Candy Crush™ or Bejewled™.

Context

When I was much younger, my family had a rule that I could only play video games for half an hour per day, except for games I built myself. This rule was a big motivating factor as I was learning Python.

At the time, my mother was occationally playing Bejewled™, and I wanted to play too. To get around my half-hour limit, I tried to build a match three game myself.

Result

Using Pygame, a game-making library for Python, I did make a game that shows a grid of colored dots, and where you can swap neighboring dots with the mouse to match three. However, the controls were clunky and I didn't know how many colors to use or how to keep score. I ended up making it get progressively harder, adding one new color after each level.

Actually playing the game was fun for a little bit, but became boring pretty quickly.
