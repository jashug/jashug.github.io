Times Table Tutor

A computer assistant to help me learn my times tables. Quick, what's 7 × 9?

Context

I worked on this project in 4th grade, when in school everyone had to learn the times tables and pass a timed test. The sort where everyone gets a worksheet with like 40 problems, and you have a minute to fill it out, and you need to get them all right to pass.

I was usually enjoying my math classes, but I wasn't very good at memorizing. Making physical flash cards seemed boring, and a lot of work to check the answers myself. At the same time, I had just started learning Python, so I decided to write a program to help me practice.

Result

I ended up creating a program which randomly generated multiplication problems like 3 × 8 = ?, waited for me to type in my answer 24, and then told me if I was correct or not before immediately moving on to the next question. This loop continued for as long as I wanted to keep practicing. When I wanted to stop, I typed q instead of answering the question.

Once I was done, the program told me how many problems I had gotten correct and how many I missed. It also kept track of how much time I spent, and computed the average time per problem. These were important to me because they were what I was being tested on in school. It also made practicing feel more like a game, with these statistics keeping score.

Skills
